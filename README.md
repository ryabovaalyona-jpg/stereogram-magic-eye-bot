# Magic Eye Bot

Telegram-бот создаёт автостереограммы из пользовательских фотографий. Пользователь
отправляет фото, выбирает скрытую форму или короткий текст и получает готовое
изображение вместе с подсказкой.

## Локальный запуск

Требуется Python 3.12.

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
export TELEGRAM_BOT_TOKEN="token-from-botfather"
python main.py
```

Переменная `BOT_TOKEN` поддерживается для совместимости, но для новых
развёртываний следует использовать `TELEGRAM_BOT_TOKEN`.

## Railway

1. Подключите этот GitHub-репозиторий к Railway.
2. В разделе Variables добавьте `TELEGRAM_BOT_TOKEN`.
3. Оставьте одну реплику сервиса.
4. Убедитесь, что старая копия бота в Replit или другом хостинге остановлена.
5. Запустите Deploy из ветки `main`.

Файл `railway.json` закрепляет команду `python main.py`, отключает засыпание
сервиса и включает до 10 перезапусков после аварийного завершения.

Если в логах появляется `Conflict: terminated by other getUpdates request`,
тот же Telegram-токен одновременно использует второй запущенный процесс.

## Надёжность

- В памяти хранится только тайл 130×130, а не исходная фотография.
- Сессия пользователя живёт один час после последнего действия.
- Одновременно хранится не более 100 сессий.
- Рендеринг выполняется вне основного асинхронного цикла.
- Отсутствующий токен завершает процесс с ошибкой, чтобы Railway выполнил
  перезапуск согласно политике сервиса.

## Проверка

```bash
python -m compileall -q main.py tests
python -m unittest discover -s tests -v
```

После деплоя вручную проверьте `/start`, загрузку фото, все кнопки форм,
собственный текст, `/random` и `/again`.
